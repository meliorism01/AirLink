# AirLink

A modular IoT platform built using Flutter and ESP32 that enables smartphone control of infrared home appliances such as air conditioners and televisions through Bluetooth communication.

## Features

- Flutter mobile application
- ESP32 firmware
- Bluetooth communication
- Infrared device control
- Modular architecture
- Extensible device support

## Tech Stack

- Flutter
- Dart
- ESP32
- Arduino Framework
- Bluetooth Low Energy (BLE)
- IRremoteESP8266
